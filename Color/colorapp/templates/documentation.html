Step 1:

Download color.css file from our Website and Extract It!

Step 2:

Add the file in your project directory.

Step 3:

Link it!

Step 4:

Use class attribute to apply color

Step 5:

1 is lightest shade and 10 is the darkest shade


bg-red-1 will apply color to the background


text-red-1 will apply red color to the text and


border-red-1 will apply 2px solid background-color


For Dotted border use border-dotted-red-1 to apply dotted border.
